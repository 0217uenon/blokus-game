# ブロックス（Blokus）— 1人 vs AI 3人

20×20（400マス）の盤面で、あなた（青）＋ AI 3人（黄・赤・緑）の4人対戦を楽しめる
ブラウザゲームです。ビルド不要・バックエンド不要の **静的サイト**（HTML / CSS / 素のJS）で、
PCでもスマホでも遊べます。

🎮 **公開URL（GitHub Pages）**: https://0217uenon.github.io/blokus-game/

## 特長

- 4段階のAI難易度（初級＝ほぼランダム ／ 中級＝貪欲 ／ 上級＝評価関数＋ブロック ／ 超上級＝モンテカルロ探索）
- 対局後の「振り返り（反省・上達アドバイス）」を棋力に合わせて自動生成
- 上達ノートをブラウザ（localStorage）に蓄積、Markdownで書き出し可能
- **スマホ／タッチ完全対応**

## 遊び方

1. 設定画面で自分の棋力と AI 3人の難易度を選び「ゲーム開始」。
2. 自分の番に、トレイからピースを選択。
3. 盤面に配置（緑＝置ける／灰＝置けない）。
4. 置けなくなったらパス。全員が置けなくなると終了し、結果と振り返りが表示されます。

### 操作

| | PC | スマホ（タッチ） |
|---|---|---|
| ピース選択 | トレイをクリック | トレイをタップ |
| 位置決め | 盤面にカーソルを合わせる（プレビュー） | 盤面をタップしてゴースト表示（**まだ置かれない**） |
| 配置 | クリック | 同じ位置をもう一度タップ、または「確定」 |
| 回転 / 反転 | R / F キー、またはボタン | 下部固定バーのボタン |
| 微調整 | — | 下部固定バーの ↑↓←→ で1マス移動 |
| 選択解除 | Esc、またはボタン | 「解除」ボタン |

## ローカルでの起動

`index.html` をブラウザで直接開くだけで動きます。スマホ実機で確認したい場合は、
同じネットワーク内に簡易サーバーを立てられます。

```bash
node test/serve.js
# → http://localhost:8731 / 同一LANなら http://<PCのIP>:8731
```

## ヘッドレステスト

ルール・AI・振り返り生成の回帰テスト（ブラウザ不要）:

```bash
node test/selfplay.js
```
